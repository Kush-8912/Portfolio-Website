## 📋 Project Overview

This is a fully static personal portfolio website with **zero frameworks** and **zero JavaScript libraries** — just raw HTML and CSS. It serves as an introduction to Kushagra Aggarwal, a tech enthusiast currently studying at Scaler School of Technology.

The portfolio includes:
- A welcoming **Hero / Home** section
- An **About Me** section
- A **Hobbies** section
- A **Skills** section showcasing current tech abilities
- A **Projects** section highlighting completed work
- A **Contact** form for getting in touch
- A responsive **Footer** with social links

---

## ✨ Features

- 🌙 **Dark Mode Toggle** — Pure CSS dark/light mode switch using a hidden checkbox and `filter: invert()` trick (no JavaScript!)
- 📱 **Responsive Design** — Mobile-friendly layout using CSS Grid and Flexbox with a breakpoint at 600px
- 🎨 **Custom Font** — Uses *Bitcount Ink* from Google Fonts for a distinctive look
- 🔗 **Smooth Scrolling** — CSS `scroll-behavior: smooth` for seamless navigation
- 🃏 **Animated Project Cards** — Hover lift effect (`translateY`) on project cards
- 📎 **Font Awesome Icons** — Social media and skill icons via Font Awesome CDN
- 💅 **Hover Animations** — Underline slide-in on nav links, scale effects on footer icons and buttons

## 🖥️ How to Run Locally

No build tools or installations required!

1. **Clone or download** this repository.
2. Open `index.html` directly in any modern web browser (Chrome, Firefox, Edge, Safari).
3. That's it — the site is fully functional with no server needed.

## 🌓 Dark Mode

Click the **🌙 moon icon** in the navigation bar to toggle between light and dark modes. This is implemented entirely in CSS using the `:checked` pseudo-class on a hidden `<input type="checkbox">` — no JavaScript required!
